# Bootcamp-en-Tecnologia-y-Gestion-de-Proyectos
Programa integral de especialización en tecnología y gestión de proyectos. Domina metodologías ágiles, liderazgo, herramientas profesionales y transformación digital. Desarrolla competencias clave para liderar proyectos complejos en entornos empresariales modernos. Forma profesionales capacitados para impulsar innovación.
